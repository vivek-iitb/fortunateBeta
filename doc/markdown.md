# Markdown 
Here is a cheatsheet for basic and extended markdown formatting:
## Basic Markdown
### Headings
# H1
## H2  
### H3
#### H4
##### H5
###### H6

### Text Styles
**Bold**
*Italics*
~~Strikethrough~~

### Lists
1. Ordered List
2. Item 2

- Unordered List
- Item 2

### Links
[Link Text](https://example.com)

### Images
Alt Text

### Blockquotes
> This is a blockquote

### Code
Inline `code`

```
Code block
```

### Tables

| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |

```javascript
var foo = 'bar';
console.log(foo);
```
